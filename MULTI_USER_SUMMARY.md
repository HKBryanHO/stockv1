## 🚀 如何使用

### 快速啟動 (Windows)
```bash
start-multi-user.bat
```

### 快速啟動 (Linux/Mac)
```bash
./start-multi-user.sh
```

### 手動啟動
```bash
# 1. 安裝依賴
npm install

# 2. 設置多用戶系統
node setup-multi-user.js

# 3. 配置環境
cp env.example .env
# 編輯 .env 文件設置 API 密鑰

# 4. 啟動服務器
npm start
```
