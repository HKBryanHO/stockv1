## 🔧 技術改進詳情

### 量化模型實現

#### 1. Jump Diffusion Model
```javascript
// 實現了完整的 Merton 跳躍擴散模型
dS = μSdt + σSdW + SdJ
```
- 跳躍頻率 λ 估計
- 跳躍幅度正態分佈
- 與連續擴散過程的結合

#### 2. Heston Stochastic Volatility
```javascript
// 隨機波動率模型
dS = μSdt + √V SdW₁
dV = κ(θ - V)dt + σ_v√V dW₂
```
- 均值回歸波動率過程
- 相關性參數 ρ
- 波動率的波動率 σ_v

#### 3. GARCH(1,1) Model
```javascript
// 條件異方差模型
σ²ₜ = ω + αε²ₜ₋₁ + βσ²ₜ₋₁
```
- ARCH 和 GARCH 係數估計
- 條件方差預測
- 波動率聚類建模
